# COC KP Helper 产品介绍手册

## 产品定位

COC KP Helper 是一个为《克苏鲁的呼唤》7版守秘人设计的轻量级案件管理与跑团辅助工具。

它不是 VTT，也不是完整规则书平台，而是一个更专注于以下场景的工作台：

- 备团时整理案件结构
- 跑团时快速查找线索和人物信息
- 团后回顾玩家行动并继续推进剧情

## 解决的问题

很多守秘人在准备和主持调查模组时，会遇到这些问题：

- 线索分散在多个文档里，跑团时查找很慢
- NPC 动机和秘密容易写散，久了会失真
- 玩家如果漏掉关键线索，守秘人临场补救压力很大
- 团后记录混乱，下次开团前需要重新梳理

COC KP Helper 的目标，就是把这些内容集中到一个统一界面里。

## 当前版本包含什么

当前可用版本已经支持：

- 案件创建、编辑、删除
- 场景管理
- 线索管理
- NPC 管理
- 场景描述库
- 线索道具管理
- 线索道具模板化预览
- 跑团记录追加与删除
- 健康检查 2.0
- 可点击并可编辑的案件关系板
- 跑团模式中的线索快速状态切换
- 本地浏览器持久化保存
- JSON 导入导出
- 全局搜索

## 产品结构

### 概览

查看当前案件的整体状态，包括：

- 核心真相
- 玩家已知
- 玩家误判
- 下一步建议
- 线索与记录统计

### 案件库

集中管理所有案件，可新增、编辑、删除和切换案件。

### 案件工作台

这是最重要的页面，用来编辑：

- 总览
- 场景
- 线索
- NPC
- 线索道具

### 跑团模式

用于开团时快速查看当前场景，并即时追加记录。

### 线索道具库

集中查看当前案件所有线索道具。

### 规则速查

提供 Keeper 高频查看的简化规则提示。

## 适合谁使用

这个版本尤其适合：

- 想从零开始整理自己模组的新手守秘人
- 想把已有模组电子化管理的守秘人
- 需要一个单机、轻量、不依赖后端的备团工具的使用者

## 当前版本的特点

### 优点

- 轻量
- 直接打开即可用
- 支持本地持久化
- 支持结构化整理信息
- 支持导入导出

### 当前限制

- 仅为单机本地版本
- 不支持多人协作
- 不包含 AI 剧情生成
- 不包含完整角色卡系统
- 不包含 VTT 地图与战斗支持

## 推荐使用方式

1. 先在“案件库”创建一个案件
2. 在“案件工作台”中补齐核心真相和结构
3. 按标签页补充场景、线索、NPC、线索道具
4. 跑团时打开“跑团模式”快速记事
5. 团后导出 JSON 做备份
